# Projeto-Ferias
Estou aprendendo a usar o GitHub e resolvi criar esse repositório desse projeto.

O projeto tem o objetivo de servir como uma agenda de treinos de musculação, rodando localmente, e possuindo algumas opções para o usuário:
1. Registrar um treino
2. Adicionar um treino
3. Consultar dados de treino
4. Sair do progama

Projeto em desenvolvimento.
